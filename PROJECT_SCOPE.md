# PROJECT_SCOPE — InteriorDesignV26

## Цель
Сайт-визитка дизайнера интерьеров с акцентом на визуальную эстетику, кейсы и заявки.

## Стек
React + TypeScript + Vite + GSAP

## MVP-страницы
1. Главная
2. Обо мне
3. Портфолио
4. Страница кейса
5. Скетчи
6. Услуги и цены
7. Контакты

## Ограничения MVP
- без CMS
- без сложных фильтров
- без мультиязычности

## Критерии готовности MVP
- все страницы работают по роутам
- минимум 3 кейса заполнены
- есть рабочий CTA на контакт
- сайт задеплоен

## Этапы
1) Базовый каркас и маршруты
2) Контентные страницы
3) Анимации
4) SEO и полировка
5) Деплой
